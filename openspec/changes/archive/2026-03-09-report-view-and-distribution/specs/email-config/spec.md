# 外发配置页规格

## 页面信息
- 路由: `/email-config`
- 权限: 仅 Admin
- 描述: 配置SMTP外发设置

## UI 规格

### 页面结构
```
+------------------------------------------+
|  标题: 邮件外发配置                       |
+------------------------------------------+
|  [表单]                                  |
|  SMTP主机: [_______________]             |
|  SMTP端口: [____] (默认25)              |
|  用户名:   [_______________]             |
|  密码:     [_______________]             |
|  发件人:   [_______________]             |
|                                          |
|  [ ] 启用外发功能                        |
|                                          |
|  [保存配置]                              |
+------------------------------------------+
```

### 表单字段
| 字段 | 类型 | 验证 |
|-----|------|------|
| smtp_host | text | 必填 |
| smtp_port | number | 必填，默认25 |
| smtp_user | text | 必填 |
| smtp_pass | password | 必填 |
| smtp_from | email | 必填 |
| is_enabled | checkbox | 默认关闭 |

### 交互
- 页面加载 → 获取当前配置
- 点击保存 → 验证并提交
- 保存成功 → 显示成功提示
- 保存失败 → 显示错误信息

## API
- GET `/api/email-config`
  - Response: EmailConfig (不含明文密码)
- PUT `/api/email-config`
  - Body: EmailConfig
  - Response: 更新后的配置
