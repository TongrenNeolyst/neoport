# 报告详情页规格

## 页面信息
- 路由: `/reports/[id]`
- 权限: 与列表页权限一致
- 描述: 只读展示报告详情和PDF附件

## 权限规则
- 无权限用户访问 → 返回403或重定向
- 只读模式，不能修改任何内容

## UI 规格

### 页面结构
```
+------------------------------------------+
|  [< 返回] 报告标题                        |
+------------------------------------------+
|  [基本信息卡片]                           |
|  股票代码: xxx                           |
|  报告类型: xxx                           |
|  分析师: xxx                             |
|  发布日期: xxx                           |
+------------------------------------------+
|  [PDF预览/下载]                          |
|  [打开PDF] [下载]                        |
+------------------------------------------+
```

### 基本信息卡片
- 股票代码
- 报告类型
- 分析师（姓名）
- 发布日期

### PDF 区域
- 只显示最新版本的PDF
- 预览组件（iframe 或 PDF viewer）
- 下载按钮

## API
- GET `/api/reports/[id]`
  - Response: Report with latest version and PDF

## 错误处理
- 报告不存在 → 404
- 无权限 → 403
- 获取PDF失败 → 显示错误提示
