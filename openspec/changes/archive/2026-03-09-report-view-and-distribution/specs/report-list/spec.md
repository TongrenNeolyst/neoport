# 报告列表页规格

## 页面信息
- 路由: `/reports`
- 权限: Admin/SA/Analyst
- 描述: 展示已发布报告列表

## 权限规则
- Admin: 查看所有已发布报告
- SA: 查看所有已发布报告
- Analyst: 查看自己submit或自己署名的已发布报告

## UI 规格

### 布局
- 使用 DashboardLayout 布局
- 侧边栏：已有菜单 + 报告查看入口

### 页面结构
```
+------------------------------------------+
|  标题: 报告列表                           |
+------------------------------------------+
|  [表格]                                   |
|  | 报告标题 | 股票代码 | 类型 | 日期 | 分析师 | 操作 |  |
|  | -------- | -------- | ---- | ---- | ---- | -- |  |
|  | ...      | ...      | ...  | ...  | ...  | [查看] |  |
+------------------------------------------+
|  [分页组件]                               |
+------------------------------------------+
```

### 表格列
| 列名 | 宽度 | 内容 |
|-----|------|------|
| 报告标题 | 30% | 报告标题文本 |
| 股票代码 | 15% | stock_code |
| 报告类型 | 15% | report_type 显示名称 |
| 发布日期 | 15% | published_at 格式化日期 |
| 分析师 | 15% | analyst 姓名 |
| 操作 | 10% | 查看按钮 |

### 交互
- 点击查看按钮 → 跳转到 /reports/[id]
- 分页切换 → 重新加载数据

## API
- GET `/api/reports/published`
  - Query: page, limit
  - Response: { data: Report[], total: number }
